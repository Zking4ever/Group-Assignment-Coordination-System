# GACS - Group Assignment Coordination System 🚀

**Managing group projects just got easier.**

From classroom assignments to team projects, **Coordination (GACS)** helps everyone stay accountable and on schedule. Create groups, assign tasks, and track progress—all in one place.

---

## 🌟 Why GACS? (The Importance)

In the world of collaborative work, coordination is often the biggest bottleneck. GACS was built to solve the "who's doing what?" problem. By providing a centralized platform for task delegation and progress tracking, GACS:
- **Ensures Accountability**: Every task has a responsible member and a deadline.
- **Reduces Chaos**: Centralizes communication and project details.
- **Boosts Productivity**: Teams stay focused on high-priority tasks with clear visual progress.
- **Simplifies Management**: Easy-to-use interface for creating groups and assignments.

## ✨ Key Features

### 👥 Group Coordination
- **Create Groups**: Start a new project space in seconds.
- **Invite Members**: Seamlessly add team members using unique **Invite Codes**.
- **Member Roles**: Manage project ownership and participation.

### 📝 Assignment Management
- **Centralized Assignments**: Organize multiple assignments within a single group.
- **Project Overviews**: Keep track of overall completion status.

### 🛠️ Task Management
- **Granular Tasks**: Break down large assignments into manageable, baby-sized tasks.
- **Ownership**: Assign specific tasks to individual team members.
- **Deadlines**: Set and monitor start dates and deadlines to keep projects on track.
- **Task States**: Track tasks through their lifecycle (To-do, In Progress, Done).

### 📈 Progress Tracking
- **Visual Indicators**: Real-time progress bars and status updates.
- **Dashboard Overview**: A clean dashboard view of all your ongoing projects and upcoming deadlines.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or later)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Zking4ever/Group-Assignment-Coordination-System.git
   cd Group-Assignment-Coordination-System
   ```

2. **Setup Backend**
   ```bash
   cd Backend
   npm install
   # The database will be automatically initialized on start
   node server.js
   ```

3. **Setup Frontend**
   ```bash
   # Open a new terminal
   cd Frontend
   npm install
   npm run dev
   ```

4. **Access the App**
   Open your browser and navigate to `http://localhost:5173`. The backend runs on `http://localhost:5000`.
